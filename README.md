## 🚀 Featured Projects

### 🏛️ Digital Civic Engagement Platform

A full-stack MERN application that empowers citizens to create petitions, participate in public polls, and engage with civic initiatives through a secure role-based system.

**Tech Stack:** React.js, Tailwind CSS, Node.js, Express.js, MongoDB

🔗 Repository: [Digital Civic Engagement Platform](https://github.com/Daameni07/digital-civic-engagement-platform)

**Key Features**

* 🔐 Role-based Authentication & Authorization
* 📢 Petition Creation and Management
* 🗳️ Public Polls with Live Voting
* 📊 Admin Dashboard
* 📄 Transparency Report Generation
* ⚡ RESTful API Integration


### 🎓 Spandan – AI-Powered Classroom Engagement Platform

An AI-powered classroom engagement platform that leverages speech recognition and LLMs to provide real-time transcription, automatic question generation, and interactive learning experiences.

Tech Stack: React.js, TypeScript, Node.js, Express.js, MongoDB, OpenAI Whisper, ASR, LLM Models

🔗 Repository: [Spandan – AI-Powered Classroom Engagement Platform](https://github.com/Daameni07/Spandan)

**Key Features**

🎙️ Real-time Speech-to-Text Transcription
🤖 AI-powered Automatic Question Generation
📄 Multi-format Processing (Voice, Text & PDF)
👨‍🏫 Teacher Dashboard & Student Portal
📊 Classroom Participation Tracking
🔐 Role-based Access Control
